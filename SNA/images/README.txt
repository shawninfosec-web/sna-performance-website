===================================================
  SNA Performance — Images Folder Guide
===================================================

Drop your images into the correct folders below.
The website will automatically use them once placed here.

---------------------------------------------------
📁 images/logo/
---------------------------------------------------
  • logo.png       → Your main SNA Performance logo (recommended: PNG with transparent background)
  • logo-white.png → White version of logo (for dark backgrounds)
  • favicon.png    → Small 32x32 or 64x64 icon (shows in browser tab)

  Once you add logo.png, update the <nav-logo> section in each HTML file
  by replacing the text logo with:
    <img src="images/logo/logo.png" alt="SNA Performance" style="height:40px;"/>

---------------------------------------------------
📁 images/gallery/
---------------------------------------------------
  Drop your project and car photos here.
  Recommended naming convention:
    project-001.jpg
    project-002.jpg
    mustang-gt350-build.jpg
    challenger-hellcat-wheels.jpg
    etc.

  Recommended size: 1200×900px or similar 4:3 ratio
  Format: JPG or WebP (smaller file size = faster loading)

  After adding photos, update contact.html (the gallery section)
  by replacing the placeholder cards with:
    <div><img src="images/gallery/your-photo.jpg" alt="Description" style="border-radius:12px;width:100%;aspect-ratio:4/3;object-fit:cover;"/></div>

---------------------------------------------------
📁 images/partners/
---------------------------------------------------
  • texas-elite-performance.png   → Texas Elite Performance logo
  • majed-tuning.png              → Majed Tuning logo

  These will be used in the Partners section on the home page and contact page.

---------------------------------------------------
TIPS:
  • Compress images before uploading at: https://squoosh.app
  • Keep file names lowercase with hyphens (no spaces)
  • WebP format is recommended for web (better quality, smaller size)
